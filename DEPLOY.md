# 🚀 Deploy Dogracy to GitHub Pages

## Step-by-Step Guide

### 1. Create GitHub Repository
1. Go to [GitHub.com](https://github.com) and sign in
2. Click **"New repository"**
3. Name it: `Dogracy`
4. Description: `Get Free Games & Accounts - A modern web app for sharing games and accounts`
5. Make it **Public**
6. Click **"Create repository"**

### 2. Upload Files
1. Click **"uploading an existing file"**
2. Drag and drop these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
   - `.gitignore`
   - `DEPLOY.md`
3. Add commit message: `Initial commit - Dogracy website`
4. Click **"Commit changes"**

### 3. Enable GitHub Pages
1. Go to your repository
2. Click **"Settings"** tab
3. Scroll down to **"Pages"** section
4. Under "Build and deployment", click **"Source"**
5. Select **"Deploy from a branch"**
6. Choose:
   - Branch: `main`
   - Folder: `/ (root)`
7. Click **"Save"**

### 4. Wait for Deployment
- GitHub will build your site (takes 1-2 minutes)
- Look for green checkmark ✅ next to the Pages section
- Your site will be live at: `https://yourusername.github.io/Dogracy`

### 5. Test Your Site
1. Visit your new URL
2. Test all features:
   - Submit a request
   - Try admin login (Dogday17 / Doggy Dogday17)
   - Test chat functionality
   - Toggle dark mode
   - Check mobile responsiveness

## 🌐 Alternative Deployment Options

### Netlify (Easiest)
1. Go to [Netlify.com](https://netlify.com)
2. Drag and drop your folder
3. Get instant deployment with custom URL

### Vercel
1. Go to [Vercel.com](https://vercel.com)
2. Import from GitHub
3. Automatic deployment on every push

### Firebase Hosting
1. Install Firebase CLI
2. Run `firebase init`
3. Deploy with `firebase deploy`

## 🐛 Troubleshooting

### Common Issues:
- **404 errors**: Make sure all files are uploaded
- **CSS not loading**: Check file names and paths
- **JavaScript errors**: Open browser console (F12)
- **Admin not working**: Clear browser cache and re-login

### Getting Help:
1. Check browser console (F12) for errors
2. Make sure all files are uploaded correctly
3. Try refreshing with Ctrl+F5 (hard refresh)
